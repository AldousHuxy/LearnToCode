Some Obscure Terms

Computer Science has accumulated a vast amount of jargon and technical language that can be overwhelming.

    • Full-Stack: Refers to a developer who is capable of working on both the frontend and backend of an application.
    • Frontend, Client, or UI: The part of a web application that users interact with directly.
    • Backend, Server, or API: An application that communicates with other computers or databases.
    • Framework/Library: A pre-built collection of code and tools that developers use to simplify the development process.
    • Compiling: The process of converting human-readable code into machine code.
    • HTML/CSS/JS: The foundational technologies for building web pages: HTML for structure, CSS for styling, and JavaScript for interactivity.

A Day in the Life of a Software Developer

The Mile High Flood District is a tech company, producing innovative software solutions that our partners rely on.

  • Frontend Developer:
    • Consuming APIs and integrating with backend services to render dynamic content.
    • Understanding page layout, responsive design, and UI/UX principles.
    • Handling user input and form validation.
  • Backend Developer:
    • Managing APIs and ensuring they are interfacing correctly with the frontend.
    • Communicating with databases to retrieve and store data.
    • Ensuring secure data transmission and storage practices.
  • DevOps Engineer:
    • Setting up and maintaining cloud infrastructure.
    • Automating deployment processes to ensure smooth releases.
    • Monitoring application performance and scalability.
  • Data Scientist:
    • Collecting and analyzing large sets of data to identify trends and insights.
    • Building predictive models using machine learning algorithms.
    • Communicating findings to stakeholders through data visualization.
  • QA Engineer:
    • Creating and executing test plans to ensure software quality.
    • Identifying and documenting bugs and issues.
    • Collaborating with developers to resolve defects.
  • Project Manager:
    • Coordinating between different teams to ensure project milestones are met.
    • Managing timelines, budgets, and resources.
    • Communicating progress and updates to stakeholders.
  • Database Administrator:
    • Managing and maintaining database systems.
    • Ensuring data security and integrity.
    • Optimizing database performance and scalability.

Did You Know?

Computer history is full of fascinating facts and milestones.

  • The ENIAC (Electronic Numerical Integrator and Computer), built in 1945, was the first general-purpose electronic computer capable of performing more calculations per second than a human could in a lifetime.
  • The term "bug" in coding originated when a real moth was found causing issues in an early computer.
  • The first message sent across the internet was "LO" in 1969, when researchers tried to send "LOGIN" but the system crashed after the first two letters.
