Logic Gates

Logic gates are the building blocks of digital circuits. They perform basic logical functions that are fundamental to digital systems.

Types of Logic Gates

  • AND Gate: Outputs true if both inputs are true.
  • OR Gate: Outputs true if at least one input is true.
  • NOT Gate: Outputs the inverse of the input.
  • NAND Gate: Outputs false only if both inputs are true.
  • NOR Gate: Outputs true only if both inputs are false.
  • XOR Gate: Outputs true if only one input is true.
  • XNOR Gate: Outputs true if both inputs are the same.

How Logic Gates Work

Logic gates take one or more binary inputs (0s and 1s) and produce a single binary output based on the type of gate and the logical operation it performs.
