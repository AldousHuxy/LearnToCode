Arithmetic Logic Unit (ALU)

The ALU is a digital circuit that performs arithmetic and logic operations.

Functions of the ALU

  • Arithmetic Operations: Addition, Subtraction, etc.
  • Logic Operations: AND, OR, NOT, etc.
  • Bit Shifting: Shifting bits left or right

How the ALU Works

The ALU takes input from the CPU registers, performs the specified operation, and returns the result to the registers.
